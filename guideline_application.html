  • Design System

流程頁面

文字
  • 文字大小請參照 Design System
  • 文字標題可使用企業色（ #00a19b primary-8 )、黑色（ #1C1C1C black-11 )
  • 備註文字照 DesignSystem , 使用 #7C7C7Cor #ACACAC
圖標
  • 線條顏色使用企業色（ #OOA19B primary-8)
  • 圖標設計方法同 Design System
版型
  • 同行銷網頁表頭，並加上流程頁名稱
  • 底色使用行銀綠（#F4F8FA primary-1)
  • 內容卡片寬度 1200px 陰影 #409D99 透明度 20％；卡片圓角 4px ；手機版則為滿版無陰影
  • 若有需分類標題，示意圖如下，企業色（#OOA19B primary-8)標題加底線
  • 表尾分為『一般版』與『警語版』
步驟表示
  • 當下與已完成步驟使用企業色（ #OOA19B primary-8)；未達進度使用( #D9D9D9 gray-5)
  • 步驟條字數，每步驟至多8個字
輸入框
  • 輸入框標題字級大小 18px 、寬度 152px ,字數8個字以上則需斷行，輸入框與標題置頂對齊
  • 輸入框長度分為兩種版本，一般欄位使用版本一，如地址等需要較長的空間則使用版本二
按鈕
  • 按鈕規範同 Design System
  • 下步按鈕律置中
  • 欄位尚未填寫完整時，按鈕底色使用（ #ACACAC gray-6 );填寫完畢時，按鈕底色使用企業色（ #OOA19B primary-8)
卡片
  • 可做一欄式、兩欄式或是三欄式，底色使用( #E1EFEF primary-2),邊框使用（ #BDE3E2 primary-3 ),不帶圓角
  • 分為『一般卡片』與『互動元件式卡片』
  • 可做一欄式、兩欄式或是三欄式，底色使用( #E1EFEF primary-2 ),邊框使用（ #BDE3E2 primary-3 ),不帶圓角
  • 分為『一般卡片』與『互動元件式卡片』
表格
  • 分為『一般表格』與『互動元件式表格』
同意事項元件
  • 分為『滾動式』與『彈跳視窗式』，可依條款類型自行選用
彈跳視窗
  • 設計方法同 Design System, 建議可使用 Fancybox 插件
  • 背景色為 #1C1C1C 70%
