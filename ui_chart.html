Chart 圖表

圖表是數值資料的圖形表示法。

  • 設計
  • 程式碼

圖表類型

(1) 長條圖

單筆資料 多筆資料
  • 顏色：#00A19B
  • 形狀：非圓角無框線
  • 保留格線
  • 多筆資料顏色選擇依序為：
    #00A19B -> #FF9900 -> #00A4EF -> #FC5187 -> #00C08B
  • 形狀：非圓角無框線
  • 保留格線

(2) 折線圖

單筆資料 多筆資料
  • 顏色：#00A19B
  • 形狀：非圓角無框線
  • 保留格線
  • 多筆資料顏色選擇依序為：
    #00A19B -> #FF9900 -> #00A4EF -> #FC5187 -> #00C08B
  • 形狀：非圓角無框線
  • 保留格線
